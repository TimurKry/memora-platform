# Монетизация Memora — Win-Win модель

> **Идея:** Memora забирает на себя цифровое оформление и координацию. Бизнес получает клиентов и меньше рутины. Семья — один понятный путь. Платформа зарабатывает на подписке и на успешных сделках, а не на «продаже горя».

---

## Ценность для каждой стороны

| Сторона | Что получает | За что платит |
|---------|--------------|---------------|
| **Семья (B2C)** | Поиск, запись, статус дела, документы, оплата в одном месте | Бесплатно (платит только за услуги бюро) |
| **Ритуальное бюро** | Лиды, white-label сайт, CRM, Cases, календарь, меньше звонков и Excel | SaaS + % с онлайн-оплат |
| **Кладбище** | Цифровая карта, слоты церемоний, меньше телефонных согласований | SaaS + партнёрская комиссия (опционально) |
| **Поставщик** | Канал продаж, заказы из экосистемы | SaaS / % с заказа |
| **Memora** | Масштабируемая выручка, данные процесса | — |

**Win-win-win:** чем больше дел закрыто через платформу, тем выгоднее всем — бюро экономит время, семья меньше страдает от бюрократии, Memora получает transaction revenue.

---

## Что «берём на себя» (операционная ценность)

Это основа премиальной подписки и доверия B2B:

| Процесс | Сейчас у бизнеса | Memora |
|---------|------------------|--------|
| Первичный контакт | Звонки 24/7, пропуски | Онлайн-запись, форма, чат (Phase 2) |
| Сбор данных и документов | Email, факс, визиты | Загрузка в Case, чеклисты, напоминания |
| Статусы для семьи | «Перезвоним» | Личный кабинет: стадия дела |
| Согласование с кладбищем | Телефон, Excel | Request → слот → подтверждение (Phase 2) |
| Счета и оплата | Счёт на бумаге, наличные | Stripe, рассрочка (Phase 2), квитанции |
| Заказ венка / урны / транспорта | Разные поставщики | Marketplace в одном Case (Phase 2) |
| Напоминания | Вручную | Email/SMS по шаблонам DE |
| Отчётность для владельца | Нет | Dashboard: лиды, конверсия, GMV |

**Позиционирование:** не «ещё один сайт», а **операционная система похоронного дела** — от первого клика до закрытия Case.

## Потоки выручки (приоритет)

### 0. Founding phase (до 5 бюро)

| Сторона | Что получает | Платит |
|---------|--------------|--------|
| **Founding-бюро (×5)** | Полный пилотный scope | **€0** — SaaS, fees, лиды |
| **Memora** | 5 референсов, Cases, product feedback | Burn; **выручка €0** |

После 5-го бюро: Starter **€99–129/мес** для новых; founding — миграция по согласованию.

---

### 1. SaaS-подписка (основа после founding)

| План | Для кого | Ориентир DE | Включает |
|------|----------|-------------|----------|
| **Starter** | Малое бюро, 1 филиал | €79–129 / мес | White-label сайт, запись, базовый CRM, 1 Case-воронка |
| **Professional** | Среднее бюро | €199–349 / мес | + Stripe, документы, staff, аналитика, скрыть «Powered by Memora» |
| **Enterprise** | Сети, холдинги | от €599 / мес | + API, SLA, custom domain, мульти-филиал |
| **Cemetery** | Кладбище как tenant | €149–299 / мес | Участки, карта, слоты, входящие запросы |
| **Supplier** | Венки, урны, транспорт | €49–99 / мес или только % | Каталог, заказы |

**Win-win:** бюро платит меньше, чем стоит 0,5 FTE на админку; Memora получает recurring revenue.

---

### 2. Комиссия с онлайн-оплат (транзакционная, Phase 1)

- Клиент платит бюро через Stripe Connect.
- **Memora application fee:** 1,5–2,5% от суммы платежа (или фикс €2–5 за транзакцию — A/B на пилотах).
- Бюро экономит на терминале/счетах и получает быстрый cash-in.

**Принцип:** fee только когда деньги реально прошли — aligned incentives.

---

### 3. Лиды с web-public (Phase 1–2)

- **web-public** — агрегатор: семья ищет бюро → переходит на white-label tenant → бронь.
- Модели (выбрать одну на пилоте):
  - **A.** Бесплатный листинг; платные «Featured» слоты в городе (€50–200 / мес).
  - **B.** CPL: €15–40 за квалифицированный лид (запись / заполненная форма).
  - **C.** Rev-share: +0,5% с оплат по лидам с платформы.

**Win-win:** малые бюро получают трафик без маркетингового бюджета; крупные платят за видимость.

---

### 4. Marketplace (Phase 2, высокий GMV)

- Поставщик → заказ внутри Case → оплата через funeral home hub.
- **Комиссия Memora:** 8–15% с заказа поставщика (зависит от категории).
- Бюро не держит склад венков; семья видит один счёт.

---

### 5. Партнёрства с кладбищами (Phase 2)

- Кладбище digitized → бюро шлёт Request через платформу.
- **Модели:**
  - Фикс за активное партнёрство (€30 / мес с каждой стороны).
  - Или % с церемонии / участка (настраивается в `partnership_agreements`, не в коде).

---

### 6. Premium-модули (Phase 2–3)

| Модуль | Цена | Ценность |
|--------|------|----------|
| SMS / WhatsApp | +€29 / мес | Напоминания семье и staff |
| AI-ассистент (черновики писем, FAQ) | +€49 / мес | Меньше времени на переписку |
| Расширенная аналитика | +€39 / мес | Воронка, cohort, GMV |
| API / интеграции | Enterprise | ERP, DATEV (Phase 3) |

## Сводная модель «где зарабатываем»

```text
                    ┌─────────────────────────────────────┐
                    │           MEMORA PLATFORM            │
                    └─────────────────────────────────────┘
         MRR ◄──────┤ SaaS (Starter / Pro / Enterprise)    │
         Fee ◄──────┤ % с онлайн-оплат (Stripe)            │
         Ads ◄──────┤ Featured / CPL с web-public          │
         GMV ◄──────┤ Marketplace commission (Phase 2)     │
         Partner ◄───┤ Cemetery / referral agreements       │
         Add-on ◄────┤ SMS, AI, analytics                   │
```

| Фаза | Главный драйвер выручки | Доля (цель) |
|------|-------------------------|-------------|
| **MVP (1–4 мес)** | SaaS + payment fee | 70% / 30% |
| **Ecosystem (5–9)** | + Marketplace + cemetery | 50% / 25% / 25% |
| **Scale (10+)** | + Enterprise + add-ons | диверсификация |

## Unit economics (черновик для пилота)

| Метрика | Гипотеза |
|---------|----------|
| ARPU (funeral home Pro) | €250 / мес |
| Payment GMV / tenant / мес | €8 000 |
| Platform fee 2% | €160 / мес |
| **Blended revenue / tenant** | **~€410 / мес** |
| CAC (assisted sales DE) | €800–1 500 |
| Payback | 3–5 мес |
| Churn (год 1) | <8% (sticky из-за Cases + CRM) |

*Цифры — для пилота; уточнить после 3–5 бюро.*

---

## Этика и доверие (важно для DE)

- **Не накручивать** скрытые сборы на семью — fee с бюро и прозрачный breakdown.
- **Не продавать данные** траурных клиентов.
- **GDPR:** EU hosting, удаление per tenant.
- Публичные цены на сайте бюро — решение tenant (см. open questions).
