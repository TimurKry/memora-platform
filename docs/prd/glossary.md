# Glossary

| Term | Definition |
|------|------------|
| **Tenant** | A paying business customer on the platform (may own multiple organizations) |
| **Organization** | A legal entity: funeral home, cemetery, supplier, etc. |
| **White-label** | Customer-facing experience branded as the organization, not Memora |
| **Multi-tenant** | Single platform instance serving many isolated tenants |
| **Partner portal** | B2B dashboard for organization staff |
| **Customer portal** | B2C experience for end users booking/paying |
| **Marketplace** | Supplier catalog with order flow and commission settlement |
| **Connect** | Stripe Connect — multi-party payment routing |
| **Plot** | Cemetery burial location unit |
| **Ceremony slot** | Scheduled time for burial/commemoration at cemetery |
| **GMV** | Gross merchandise volume — total payment flow through platform |
